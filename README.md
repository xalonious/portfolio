# portfolio

A modern web portfolio.  
Built with Next.js, React, TypeScript, Tailwind CSS, and Framer Motion.

## Overview

this is my personal site, built to bring together my story, projects, tech stack, live Discord presence, and contact links in one responsive web experience.

## Features
- Responsive landing page with animated hero, smooth scrolling, and page transitions
- Project showcase with featured projects and a dedicated projects page
- Interactive tech stack radar with category filtering and hover details
- Live Discord and Spotify presence powered by the Lanyard API
- Contact form with validation, rate limiting, spam checks, and Discord webhook delivery
- Custom preloader, scroll progress, magnetic buttons, and scramble text effects
- Hidden Konami chat feature
